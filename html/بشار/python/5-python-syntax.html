المسافات في البايثون

تؤثر المسافات عند كتابة الكود في لغة البايثون ولكن في بعض اللغات الاخري يتجاهلها المترجم Compiler

- لتجنب ظهور خطاء Syntax Error يجب وضع مسافة بدائيه مثل الكود التالي :

    • if 10 > 5:

    print("Ten is greater than Five!")

    if 10 > 5:

    print("Ten is greater than Five!")

    -في حالة لم تضع المسافه البدائيه سوف يظهر خطاء Syntax Error مثل الكود التالي:

  • Syntax Error

    • if 10 > 5:

    print("Ten is greater than Five!")

    - عدد المسافات متروك لك كمبرمج، والاستخدام الأكثر شيوعا هو أربعة، ولكن يجب أن يكون واحد على الأقل.

    • if 10 > 5:

    print("Ten is greater than Five!")

    if 100 > 5:

    print("Ten is greater than Five!")

    - يجب عليك استخدام نفس عدد المسافات في نفس كتلة التعليمات البرمجية، وإلا فسوف يمنحك خطأ Syntax Error:

  • Syntax Error

    • if 10 > 5:

    print("Ten is greater than Five!")

    print("Hello World!")

    كيفية عمل تعليق في البايثون

    التعليق هو سطر يتجاهله المترجم Compiler يساعد المبرمج في تذكر مهمة هذا الكود

    لأنشاء تعليق يجب ان يبدأ ب # وسوف يتجاهله المترجم Compiler

    • #print Hello World

    print("Hello World!")

    يمكن وضع التعليق امام الكود

    • print("Hello World!") #print Hello World

    لعمل تعليق لأكثر من سطر يجب ان تبدأ ب ' ' ' وتنتهي ب ' ' ' مثل المثال التالي:

    • ''' print Hello World
    print Hello World
    print Hello World '''

    print("Hello World!")
