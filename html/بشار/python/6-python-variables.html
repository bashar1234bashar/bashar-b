التعرف علي المتغيرات Variables


    طريقة انشاء متغير في البايثون طريقة معرفة نوع المتغير حالة الأحرف

طريقة انشاء متغير في البايثون

المتغيرات عبارة عن مكان يتم تخزين فيه البيانات

في البايثون يتم انشاء متغير في اللحظه التي تقوم فيها بتعيين قيمه لها ولا تحتاج الي تعريف نوع المتغير مثل بعض اللغات الاخري

    • x = 10

    y = "Hello"

    print(x)

    print(y)
    Result: 
     10
 Hello

    لا يلزم الإعلان عن المتغيرات مع أي نوع ، ويمكن حتى تغيير نوع المتغير بعد تعيينه.

    • x = 10 # int x هنا نوع المتغير

    x = "Hello"# str نص x الأن اصبح نوع المتغير

    print(x)

    # سوف نحصل علي اخر قيمة للمتغير
    Result: 
     Hello

    إذا كنت ترغب في تحديد نوع بيانات المتغير، فيمكن القيام بذلك كما في الكود التالي:

    • x = str(50) # str أنه نص x هنا قمنا بتعريف نوع المتغير

    y = int(50) # int أنه عدد صحيح y هنا قمنا بتعريف نوع المتغير

    z = float(50) # float أنه عدد عشري z هنا قمنا بتعريف نوع المتغير

    s = bool(0) # (True او 1 False 0) bool أنه s هنا قمنا بتعريف نوع المتغير

    print('x =' ,x)

    print('y =' ,y)

    print('z =' ,z)

    print('s =' ,s)
    Result: 
     x = '50'
 y = 50
 z = 50.0
 s = False

    طريقة معرفة نوع المتغير

    يمكنك معرفة نوع المتغير من خلال استخدام دالة ()type كما في الكود التالي:

    • x = "Python"

    y = 50

    z = 50.5

    s = True

    print('x =' ,type(x))# x لمعرفة نوع المتغير

    print('y =' ,type(y))# y لمعرفة نوع المتغير

    print('z =' ,type(z))# z لمعرفة نوع المتغير

    print('s =' ,type(s))# s لمعرفة نوع المتغير

    Result: 
     x = str
 y = int
 z = float
 s = bool

    حالة الأحرف

    البايثون حساسه اتجاه الأحرف اختلاف اللغات الأخري , في البايثون اذا كان الحرف capital حرف كبير اختلاف small حرف صغير كما في المثال التالي:

    • a = "Python"# str نص a هنا نوع المتغير

    A = 50# int عدد صحيح A هنا نوع المتغير

    print('a =' ,a)

    print('A =' ,A)
    Result: 
     a = Python
 A = 50
