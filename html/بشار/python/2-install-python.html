تجهيز بيئة العمل للبرمجة بلغة بايثون

سوف نقوم بتحميل آخر إصدار متوفر من لغة بايثون و نسخة مجانية من برنامج Anaconda لبناء و تشغيل كود البايثون على الكمبيوتر. سبب إستخدامنا لبرنامج Anaconda هو أنه سهل الإستخدام ويحتوي علي عدة محررات ومن اهمها vs code , jupyter notebook , python , PyCharm

خطوات تحميل و تنصيب لغة بايثون.

1- الدخول علي اللينك التالي لتنزيل الادوات اللازمه للغة البايثون من الموقع الرسمي من هنا
2- قم بتحميل اخر أصدار متوافر.

    • 3- أنقر على Save File لكي يبدأ التحميل.

    • 4- بعد أن يتم التحميل ، قم بفتح الملف لتبدأ بتنصيب ادوات البايثون.
    قم بالنقر علي زر الفأره الأيمن، ثم أنقر علي Run as administrator

    • 5- قم بوضع علامة صح على الخيار Install launcher for all users و علامة صح على الخيار Add Python 3.6 to PATH
    ثم أنقر على الزر Install Now ليبدأ تنصيب الأدوات على الجهاز.

    • 6- بعد أن يكتمل تنصيب الأدوات بنجاح, إضغط على Close.

    • لتأكد من ان لغة البايثوم مثبته علي جهازك يمكنك الذهاب الي Command Prompt (cmd) وكتابة الامر التالي سوف تقوم بتشغيل البايثوم ومعرفة الاصدار.

    • يمكنك تجريب امر الطباعة في Command Prompt (cmd) من خلال كتابة الامر التالي print("Hello World")
